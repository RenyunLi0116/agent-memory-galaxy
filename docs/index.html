Collect, distill, merge, encrypt, view.

Contributor machines write private fragments. An aggregator merges shared entities, injects live presence, and optionally publishes an encrypted viewer shell.

01Collect

Scan reviewed `agent_memory.md` notes from explicit project roots.

02Distill

Extract safe structured metadata from agent sessions without copying raw text.

03Merge

Connect projects through shared files, datasets, models, tools, and servers.

04Encrypt

Publish ciphertext only when deploying an encrypted viewer shell to Pages.

05View

Search, filter, zoom, and inspect the resulting memory galaxy.

Public framework. Private memory.

GitHub Pages is not private access control. Public pages carry the framework and fake demo data. Real fragments and plaintext graphs belong in a private hub or local machine.

Contributor

Writes `fragments/<machine>.json` in a private hub. Does not need encryption passwords and should not touch `docs/galaxy/`.

Aggregator

Merges all fragments, builds local `standalone.html`, and optionally creates `docs/galaxy/graph.enc.json` with strong passwords.
