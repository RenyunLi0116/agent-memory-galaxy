What It Does

Agent Memory Galaxy turns scattered work traces into a durable graph: hand-written memory notes, safe native-session metadata, fragments pushed by other machines, and live agent presence.

01

Collect

Find agent_memory.md files, optional Claude memory, and contributor fragments.

02

Normalize

Convert entries, facts, files, datasets, models, and projects into graph nodes.

03

Merge

Connect machines through shared entities and project references.

04

Encrypt

Optionally publish only ciphertext for browser-local unlock.

05

View

Explore the work graph as an interactive 3D memory galaxy.

Privacy Model

Plaintext is local

Generated graph.json and standalone.html are gitignored and meant for local/private use.

Fragments are private

Multi-machine fragments can contain sensitive work context. Keep them in a private hub.

Pages serves ciphertext

If you publish online, use docs/galaxy/ with encrypted graph data and strong passwords.

Install the Skill

The skill lets another coding agent set up the repo, connect a machine, run contribution or aggregation workflows, and enforce the privacy checklist.
